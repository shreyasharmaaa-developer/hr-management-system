# HR Management System

A complete, responsive HR Management System built with pure HTML, CSS, and JavaScript.

## Pages Included

| Page | File | Description |
|------|------|-------------|
| Dashboard | index.html | Overview stats, department chart, activities |
| Employees | pages/employees.html | Employee list, add/edit/delete |
| Attendance | pages/attendance.html | Daily attendance tracking |
| Payroll | pages/payroll.html | Salary management & payslips |
| Leave Management | pages/leave.html | Leave requests & approvals |
| Recruitment | pages/recruitment.html | Job postings & candidates |

## How to Run

Simply open `index.html` in any modern browser — no server required.

## Features

- Responsive sidebar navigation
- Animated stat counters
- Search/filter in tables
- Modal forms for data entry
- Toast notifications
- Mobile responsive design

## Tech Stack

- HTML5, CSS3, Vanilla JavaScript
- Font Awesome icons (CDN)
- No frameworks, no build tools required
